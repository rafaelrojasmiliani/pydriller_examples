# PyDriller Examples

This repository contains minimal examples for working with
[PyDriller](https://pydriller.readthedocs.io/en/latest/index.html).

## Repository structure

- `examples/`
  - `basic/`: Small, focused scripts for first-time usage patterns.
  - `commit_history/`: Examples that focus on commit traversal and filters.
  - `metrics/`: Examples that compute simple repository metrics.
- `docs/`: Notes and walkthroughs for the examples.
- `scripts/`: Helper scripts (future automation, setup, or data prep).

Each example directory includes a short README to describe intended content.
